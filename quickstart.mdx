This guide walks you through cloning the repo, installing dependencies, and launching the game server locally.

## Prerequisites

Before you begin, make sure you have the following installed:

- **Node.js** (v14 or later) — [nodejs.org](https://nodejs.org)
- **Git** — [git-scm.com](https://git-scm.com)

## Setup

<Steps>
  <Step title="Clone the repository">
    Clone the Lucky Roll Dice repo from GitHub:

    ```bash
    git clone https://github.com/dumitrusf/roll-Dice.git
    ```
  </Step>

  <Step title="Navigate to the project directory">
    ```bash
    cd roll-Dice
    ```
  </Step>

  <Step title="Install dependencies">
    Install the two runtime dependencies — `express` and `reset-css`:

    ```bash
    npm install
    ```
  </Step>

  <Step title="Start the server">
    Run the Express static file server:

    ```bash
    npm start
    ```

    The server starts on port **3000** by default. You will see no output in the terminal — this is expected.

    <Note>
      To use a different port, set the `PORT` environment variable before starting:

      ```bash
      PORT=8080 npm start
      ```

      Then open `http://localhost:8080` instead.
    </Note>
  </Step>

  <Step title="Open the game in your browser">
    Navigate to:

    ```text
    http://localhost:3000
    ```

    You should see the Lucky Roll Dice game UI with two player panels and the dice in the center.
  </Step>

  <Step title="Initialize a new game">
    Click **NEW GAME** to reset all scores and set Player 1 as the active player. The red dot next to a player's name indicates whose turn it is.

    <Tip>
      Always click **NEW GAME** before starting play. Without it, the roll and hold buttons have no active game state to work with.
    </Tip>
  </Step>

  <Step title="Play">
    With the game initialized, use the two action buttons on each turn:

    - Click **ROLL DICE** to roll the die. Rolling a 2–6 adds points to your current score. Rolling a **1** loses your current score and ends your turn.
    - Click **HOLD** to bank your current score into your global total and pass the turn to your opponent.

    The first player to reach **100 global points** wins.
  </Step>
</Steps>
